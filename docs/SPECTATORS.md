## The Definitions

### Glass-Wall

A live observable boundary around the arena that exposes block activity, tribunal actions, scoring traces, and cryptographic receipts to external spectators — without requiring them to participate in execution.

### Spectator

An external verifier who can independently inspect, replay, dispute, and confirm the validity of the tribunal's process and outputs.

### The Distinction

Spectators are not judges. They are **witnesses with verification power.**

Judges can be accused of bias. Witnesses checking open evidence create a stronger legitimacy layer.

All without entering the arena. All without trusting anyone.

## Challengeable Transparency

The Glass-Wall is not just transparency. It is **challengeable transparency.**

```
Transparency:              "Here's what we did"
Challengeable transparency: "Here's what we did — prove us wrong"
```

Every claim is disputable:

| Claim | How to challenge |
|-------|-----------------|
| "This pair scored 1.0" | Re-run the scoring function yourself |
| "This was mined at 14:23 UTC" | Check the Hedera anchor timestamp |
| "The lineage is scout → peeta → katniss" | Trace the attempt IDs in the chain |
| "Cost-per-honey was $0.06" | Sum the energy receipts, divide by honey count |
| "28 nodes participated" | Count unique node IDs in the block's attempts |
| "This dataset is defendable" | Run the 5-proof audit yourself |

If you can't dispute it, it stands. If you CAN dispute it and the math holds — it's proven. Either way, the Glass-Wall wins.

## The Five Layers of Trust

```
Layer 1: DETERMINISTIC SCORING
  The math is the same for everyone.
  Same input + same function = same score.
  No opinion. No model. No judge. Math.

Layer 2: ELIMINATION PRESSURE
  What survives the tribunal earned survival.
  Not selected by humans. Selected by pressure.
  The process is the proof.

Layer 3: LIVE TRANSPARENCY
  The Glass-Wall shows everything in real-time.
  Not a report about what happened.
  The thing itself, happening, visible.

Layer 4: INDEPENDENT VERIFICATION
  Spectators replay, dispute, confirm.
  No trust in Swarm & Bee required.
  Trust the math. Verify the proof.

Layer 5: IMMUTABLE RECEIPT
  Hedera anchor. Merkle root. Timestamp.
  Can't be changed. Can't be deleted. Can't be faked.
  The record exists forever.
```

Each layer reinforces the others. Remove one and the system still functions — but all five together make it **institutional grade.**
